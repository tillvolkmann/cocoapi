# COCO API
Fork of COCO API with extended functionality.

### Installation
pip install git+https://github.com/tillvolkmann/cocoapi.git#subdirectory=PythonAPI

### Requirements
Following requirements are only needed for respective data set conversions:
* VOC: lxml
* tfrecord: tensorflow
